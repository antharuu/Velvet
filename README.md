# 🎀 **Velvet** 🎀

**Velvet is a template engine for PHP, it is easy to use, light, fast and customizable.**

*It is strongly inspired by one of my previous projects, Mew, itself inspired by Pug.*

### Why create a template engine?

Because it makes life easier, writing HTML is laborious, and even more so to integrate PHP. It avoids a lot of code
repetition and makes everything very readable.

### Why not use Pug, Blade or Twig or others?

Because Blade & Twig use an HTML base, certainly they simplify the PHP integration, but do not change anything to the
fact that the whole is laborious and quickly unreadable.

As for Pug, it is mainly JS oriented (even if there is a PHP version) and above all is very old in its logic and lacks
features.

And all of them are not customizable enough for my taste, I like that everything is simple to customize, and that
everyone can add or remove features (and why not integrate them officially).

### Any versions of Velvet planned for other languages?

Eventually, in JS, this is not a priority at all, the most important thing is to have a stable version of VelvetPHP.

Let me know if you want other versions or features.

### Is it an open-source project?

Of course, everyone is free to participate. I am looking for collaborators with the same vision of the project as me.
